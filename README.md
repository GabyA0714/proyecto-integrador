# Proyecto Integrador

Este proyecto corresponde al desarrollo de un sistema como parte del Proyecto Integrador de 3er año terciario.

## Tecnologías utilizadas

- Frontend: React / HTML / CSS / JavaScript  
- Backend: Node.js / Express  
- Base de datos: MySQL / SQL Server / Firebase  
- Control de versiones: Git & GitHub  

##  Estructura del proyecto
/frontend
/backend
/database
/docs

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/GabyA0714/proyecto-integrador.git

Backend
cd backend
npm install
npm start

Frontend
cd frontend
npm install
npm start

Configuración

Crear archivo .env en el backend:

PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=*****

Funcionalidades
Registro e inicio de sesión
Gestión de datos
ABM (altas, bajas, modificaciones)
Validaciones frontend/backend

 Notas

Proyecto académico con fines educativos.
